# Iteration Progress - 2026-05-05

## Scope

Initial framework setup for a furniture showcase website with CMS-driven content and contact workflow.

- Payload CMS integrated with PostgreSQL adapter.
- Resend email adapter integrated in Payload config.
- shadcn/ui initialized and base UI components added.
- CMS models configured:
  - `users`
  - `media`
  - `furniture`
  - `site-settings` (global)
- Furniture slug generation implemented as:

- Homepage implemented:
  - Hero video banner
  - Marketing content block (`Low price, customisable, all included services`)
  - Furniture cards (image, price, inquiry)

  - `/category/[roomCategory]`

  - `/[roomCategory]/[furnitureCategory]/[name]/[size]`

  - `/contact-us`

- Payload routes integrated:
  - `/admin/[[...segments]]`
  - `/api/[...slug]`
- Docker development setup added:
  - `Dockerfile.dev`
  - `docker-compose.yml` with `postgres` and `app` services
